# James-Bond-Sync-2
Universal Clipboard Sync 

## Architecture for the app
 -Top App bar for toggling between pages

 2 major pages:

 -Device page(main page): -sync status (2/3 online)
                          -Paired Devices(heading)
                          -list the paired devices(one dot - signalling online/offline | toggle key (pair))
                          -(+)Pair a new device

 -History page: -Shows the copied texts(with timestamp + deviceName + copy/paste)   
                -also highlights the ACTIVE text (that would paste on Ctrl+c)
